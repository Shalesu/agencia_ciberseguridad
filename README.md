# agencia_ciberseguridad
## 📌 Introducción
Este proyecto simula una agencia digital mediante la creación de un sitio web de presentación. Se aplicaron conocimientos de **HTML, CSS y JavaScript/PHP**

## 🚀 Funcionalidades
- Estructura HTML y CSS bien definida.
- Sección dinámica con formulario de contacto (JavaScript)
- Navegación entre secciones mediante botones y enlaces.
- Diseño simple y responsivo.
